Tags

As tags HTML são elementos usados para estruturar e formatar o conteúdo em uma página da web.
As Principais São:

  • p
  • ul
  • h1
  • ol
  • title
  • span

Atributos

(voltar)

Os atributos HTML são valores adicionados às tags HTML para fornecer informações extras sobre um elemento ou para especificar comportamentos específicos. Eles são usados para modificar ou fornecer detalhes adicionais sobre como um elemento deve ser apresentado ou comportar-se.

Clases

(voltar)

Classes HTML são atributos fundamentais que podem ser adicionados a elementos HTML para agrupá-los e aplicar estilos de forma mais eficiente usando CSS (Cascading Style Sheets). Elas oferecem uma maneira de identificar e segmentar conjuntos específicos de elementos semelhantes em uma página da web. Ao atribuir uma classe a elementos HTML, você cria uma associação semântica entre eles, permitindo que sejam estilizados de maneira uniforme ou diferenciada com base em critérios de design. Essa abordagem promove uma manutenção mais fácil e consistência visual em todo o site, facilitando a gestão e aprimoramento da experiência do usuário.

Curiosidades HTML

(voltar)
  1. HTML não é uma linguagem de programação: Apesar de ser frequentemente mencionada como uma linguagem de programação, HTML (HyperText Markup Language) é, na verdade, uma linguagem de marcação usada para estruturar o conteúdo em uma página da web.
  2. A primeira versão do HTML foi lançada em 1991: O HTML foi proposto por Tim Berners-Lee, o inventor da World Wide Web, e a primeira especificação foi lançada em 1991. Desde então, o HTML evoluiu para várias versões, com o HTML5 sendo a mais recente.
  3. HTML5 trouxe muitos recursos novos: O HTML5 introduziu uma série de recursos novos e avançados, como vídeo e áudio nativos, elementos semânticos ,suporte a canvas para gráficos dinâmicos, entre outros.
